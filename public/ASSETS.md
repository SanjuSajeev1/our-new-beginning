# Add your media files

Place these files in `public/`:

- `mmuah.mp4`
- `us.jpg`

Place these files in `public/images/`:

- `emotional1.jpg` ... `emotional10.jpg`
- `confident1.jpg` ... `confident10.jpg`
- `love1.jpg` ... `love10.jpg`
- `fun1.jpg` ... `fun10.jpg`
- `playful1.jpg` ... `playful10.jpg`

The app uses these local paths first and falls back to themed online placeholders when an image is missing.
